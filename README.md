# 3-Stage Pipeline (SystemVerilog)

A 3-stage pipelined processor design written in SystemVerilog.

## Pipeline Stages
1. **Fetch**
2. **Decode**
3. **Execute**

## Modules Designed
- PC
- Instruction Memory
- Register File
- Immediate Generator
- ALU
- Control Unit
- Data Memory
- Pipeline Registers

## Simulator
Vivado
